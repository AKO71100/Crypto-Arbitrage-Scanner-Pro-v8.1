Профессиональный инструмент для мониторинга межбиржевого арбитража на рынке криптовалютных фьючерсов. Сканер в реальном времени анализирует цены на ведущих площадках и предоставляет статистику по стабильности спреда.

✨ Основные возможности
Мультибиржевой мониторинг: Поддержка API Binance, Bybit, OKX, Gate.io, BingX и Hyperliquid.

Анализ истории (K-lines): При обнаружении возможности сканер автоматически подгружает минутные свечи с бирж за последний час для проверки "реальности" и долговечности спреда.

Интеллектуальный рейтинг: Система оценки возможностей (🟢 SUPER, 🟡 GOOD, 🔴 RISKY) на основе размера профита и торгового объема.

Локальная база данных: Хранение истории спредов в SQLite для расчета метрик (Min/Max за 10 мин, Delta, количество сигналов за час).

Фильтрация: Настройка минимального профита, объема торгов (24ч) и выборки монет по капитализации через CoinGecko.

🛠 Технологический стек
Язык: Python 3.x

Интерфейс: Tkinter (ttk)

База данных: SQLite3

Параллелизм: Threading, ThreadPoolExecutor (для быстрой загрузки цен и свечей)

Сеть: Requests с кастомными адаптерами повторных попыток (Retry)

🚀 Быстрый старт
Установите зависимости:

Bash
pip install requests
Запустите приложение:

Bash
python scanner_ui.py
⚙️ Настройки
Min Spread (%): Минимальный чистый профит (после вычета комиссий) для отображения в таблице.

Min Vol ($): Фильтр по минимальному суточному объему торгов для исключения неликвидных пар.

Top Coins: Количество топовых монет по капитализации (из CoinGecko) для сканирования.

Hist Window: Временной интервал в минутах для расчета локальной статистики (Min/Max).

📊 Интерфейс таблицы
Net: Чистая прибыль в % (уже учитывает средние комиссии бирж).

Min10/Max10: Минимальное и максимальное значение спреда за последние 10 минут.

Delta: Разница между Max и Min (показатель волатильности спреда).

Hit1h: Количество раз, когда спред был выше заданного порога за последний час.
